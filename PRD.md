# 파라다이스 계산기 — PRD
> "원하는 삶에서 출발하는 역산 FIRE 시뮬레이터"  
> 최종 버전 · 2026.03

---

## 1. 배경 및 문제 정의

### 1-1. 배경

미래의 자산 설계와 경제적 자유를 꿈꾸지만 방법을 찾지 못한 사람들을 위한 프로젝트.  
월 저축액과 인플레이션을 반영한 **개인 맞춤 자산 청사진**을 시각적으로 제시한다.

### 1-2. 기존 낙원계산기의 문제

| 문제 | 설명 |
|------|------|
| 숫자만 표시 | 그래프 없음 → 복리 효과 체감 어려움 |
| 변수 비교 없음 | 시나리오 실험 불가 |
| 공유·저장 불가 | 바이럴 및 재방문 없음 |
| 순방향 계산 | "얼마 모으면 돼?" 대신 "어떤 삶을 원해?"에서 출발해야 함 |

---

## 2. 제품 목표

**희망 월 인출액**을 입력하면, 그것을 이루기 위한 필요 자산과 월 저축액을 역산하여 구체적인 경로를 제시한다.

```
입력 (사용자가 원하는 삶)           출력 (역산 결과)
──────────────────────────          ──────────────────────
희망 월 인출액   300만원     →      필요 은퇴 자산   9억원
은퇴 희망 나이    55세        →      월 필요 저축액  187만원
현재 나이         35세        →      현재 페이스 은퇴  59세
현재 자산      5,000만원      →      목표까지          D-20년
```

## 5. UI Flow (현재 구현)

```
[대시보드 (단일 페이지)]
        │
        │  "수정" 버튼 클릭 (우상단 pill 버튼)
        │  → 플로팅 사이드바 슬라이드인
        ↓
[조정 사이드바]
        │
        │  슬라이더 조작 → debounce 300ms → 즉시 재계산
        │  차트·카드 실시간 업데이트
        ↓
[대시보드 (동일 화면)]
```

URL 파라미터로 현재 입력값을 인코딩하여 공유 링크 생성 가능.

## 6. 페이지 구조

### 6-1. Topbar (고정)

- 높이 56px, 페이지 상단 고정
- 좌: 로고 (`P`) + 히어로 메시지 ("안녕하세요, 미학님 / 목표 자산의 N% 모았어요")
- 우: 수정 버튼, 공유 버튼, 아바타

### 6-2. 결과 카드 (4종)

| 카드 | 표시 내용 | 강조색 |
|------|-----------|--------|
| 필요 은퇴 자산 | FIRE 달성에 필요한 총 자산 (억) | `--gold` (#81CE16) |
| 월 필요 저축액 | 목표 달성을 위한 역산 저축액 | `--primary` (#111111) |
| 현재 페이스 은퇴 | 지금 저축 페이스로 은퇴 가능한 나이 | 상황별 |
| 목표까지 D-N년 | 목표 은퇴까지 남은 햇수 | 상황별 |

### 6-3. 자산 성장 로드맵 차트

- **타입**: Chart.js Line Chart
- **X축**: 나이 (현재 나이 ~ 최대 나이 90세)
- **Y축**: 자산 (억원)
- **라인 색상**: `PERSON_COLORS[0]` = `#81CE16` (기본 사용자)
- **라인 굵기**: 2px, `borderCapStyle: 'round'`
- **수직 보조선 2개**:
  - 목표 은퇴 나이 (연하게, `rgba(0,0,0,0.07)`)
  - 은퇴 예상 나이 (조금 진하게, `rgba(0,0,0,0.12)`)
- **툴팁**: hover 시 나이·자산·은퇴 라벨 표시 (해당 나이에서만)

### 6-4. 조정 사이드바

```

──── 원하는 삶 ────

  은퇴 희망 나이   슬라이더 (40~70 세)
```

## 7. 기능 요구사항 (MoSCoW)

### 🔴 Must Have — 완료

#### F-01 역산 FIRE 계산기
- 희망 월 인출액 + 안전 인출률 → 필요 은퇴 자산 역산
- 필요 자산 + 현재 자산 + 수익률 → 월 필요 저축액 역산
- 현재 저축 페이스 기준 FIRE 달성 나이 계산
- 슬라이더 조작 → 300ms debounce → 즉시 재계산

#### F-02 자산 성장 시각화
- 연령별 자산 라인 차트 (Chart.js)
- 은퇴 목표 / 예상 시점 수직선 표시
- hover 툴팁: 나이·자산·은퇴 구분 정보

#### F-03 인플레이션 반영
- 명목 자산 기준 계산
- 인플레이션 배너: "실질 구매력 N억 (현재가치)" 표시
- 기본값 2.5%, 사용자 조정 가능

#### F-04 페르소나 프리셋
- 사이드바에서 안정형 / 공격형 1클릭 적용
- 적용 후 각 슬라이더 개별 조정 가능

#### F-07 URL 공유
- 입력값 → URL 파라미터 인코딩
- 공유 링크 복사 기능

## 8. 계산 로직 요약

### FIRE 필요 자산
```
fireTarget = (희망 월 인출액 × 12) / (안전 인출률 / 100)
```

### 역산 월 저축액 (목표 나이까지 FIRE 달성)
```
N = 목표 나이 - 현재 나이
r = 연 수익률 / 12 (월 이율)
g = 저축 증가율 / 12 (월 증가율)

미래가치(현재 자산) = currentAssets × (1 + r)^(N×12)
필요 저축 미래가치 = fireTarget - 미래가치(현재 자산)
→ 월 저축액 역산 (증가율 반영 FV 공식)
```

### 은퇴 예상 나이 (현재 페이스 기준)
```
현재 저축액으로 매년 시뮬레이션 → 자산이 fireTarget 초과하는 첫 나이
```

### 인플레이션 반영 실질 구매력
```
실질가치 = 명목자산 / (1 + infRate/100)^경과연수
```
