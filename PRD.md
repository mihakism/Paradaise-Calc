# 파라다이스 계산기 v2.0 — PRD
> "원하는 삶에서 출발하는 역산 FIRE 시뮬레이터"
> Version: 2.0 Draft · 2026.03

---

## 1. 배경 및 문제 정의

### 1-1 배경
미래의 자산 설계와 경제적 자유를 꿈꾸지만 방법을 찾지 못한 사람들을 위한 프로젝트.
월 저축액과 인플레이션을 반영한 개인 맞춤 자산 청사진을 제시한다.

### 1-2 문제 정의
기존 낙원계산기의 4가지 핵심 문제:
- 숫자만 표시, 그래프 없음 → 복리 효과 체감 어려움
- 변수 비교 기능 없음 → 시나리오 실험 불가
- 결과 공유·저장 불가 → 바이럴 및 재방문 없음
- 순방향 계산 구조 → "얼마 모으면 돼?" 대신 "어떤 삶을 원해?"에서 출발해야 함

---

## 2. 제품 목표

희망 월 인출액을 입력하면, 그것을 이루기 위한 필요 자산과 월 저축액을 역산하여 구체적인 경로를 제시한다.

```
입력 (사용자가 원하는 삶)          출력 (역산 결과)
──────────────────────────         ──────────────────────
희망 월 인출액   300만원     →     필요 은퇴 자산   9억원
은퇴 희망 나이   50세        →     월 필요 저축액   187만원
현재 나이        35세        →     현재 페이스 은퇴  52세
현재 자산        3,000만원   →     목표까지          D-15년
```

## 3. 타겟 유저

재테크에 관심은 있지만 "지금 시작해도 늦지 않을까" 걱정하는 **사회초년생 ~ 40대** 개인 투자자

## 5. UI Flow

```
[온보딩 페이지]
    │
    │  입력 완료 후 "내 낙원 보기" CTA 클릭
    │  → 페이지 전체 페이드아웃 → 대시보드 페이드인
    ↓
[대시보드 페이지]
    │
    │  우상단 ⚙️ 설정 버튼 클릭
    │  → 배경 블러 + 설정 모달 슬라이드인
    ↓
[설정 모달]  (대시보드 위 오버레이, 별도 페이지 이동 없음)
    │
    │  "다시 계산하기" 클릭
    │  → 모달 닫힘 + 그래프 즉시 업데이트
    ↓
[대시보드 페이지]
```

## 6. 페이지별 설명

### 6-1 온보딩 페이지

**목적:** 사용자의 삶의 목표 4가지를 입력받는다.

**구조:** 5개 섹션, 각 섹션 풀스크린(100vh). savor.it처럼 스크롤 시 배경이 전환되는 시각적 효과 적용. 단, 실제 진행은 Enter 키 또는 "다음 →" 버튼으로만 가능 (휠·스와이프로 섹션 이동 불가).

```
Section 1  "지금 몇 살이세요?"          현재 나이 입력
Section 2  "매달 얼마를 저축하나요?"    월 저축액 입력
Section 3  "언제 은퇴하고 싶으세요?"   은퇴 희망 나이 입력
Section 4  "매달 얼마가 필요하세요?"   희망 월 인출액 입력
Section 5  결과 프리뷰 + 페르소나 선택 + CTA
```

**섹션별 배경 (새벽 → 낙원으로 변화):**
```
Section 1  새벽  #0F1117 → #1A1D2E
Section 2  일출  #1A1D2E → #1E3A5F
Section 3  노을  #1E3A5F → #0D4F3C
Section 4  낙원  #0D4F3C → #0FAD77
Section 5  도달  #0FAD77 → #1E4FBF
```

**Section 5 구성:**
- 입력값 기반 실시간 프리뷰 문구 표시
  > "35세부터 시작하면 50세에 낙원 도달 가능 🌴"
- 페르소나 선택: `[ 안정형 ]` `[ 공격형 ]` → 수익률·저축증가율 자동 세팅
- CTA 버튼: "내 낙원 보기 →" (호버 시 글로우 효과)

---

### 6-2 대시보드 페이지

**목적:** 역산 계산 결과를 시각적으로 보여주고, 변수 조정으로 시나리오를 실험하게 한다.

**레이아웃 (데스크탑):**
```
┌─────────────────────────────────────────────────────┐
│  Header: 로고  +  "내 낙원까지 D-15년"  +  ⚙️ 설정   │
├─────────────────────────────────────────────────────┤
│  결과 카드 3종 (1/3씩 가로 배열)                      │
│  [ 필요 자산 9억 ]  [ 월 저축 187만 ]  [ 은퇴 50세 ]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  자산 성장 그래프  (height: 400px)                   │
│  - 은퇴 시점 수직 점선 + "🌴 낙원 도달" 레이블        │
│  - 호버 시 툴팁: 연도 / 자산액 / 은퇴까지 N년         │
│                                                     │
├─────────────────────────────────────────────────────┤
│  시나리오 A/B 비교 토글  +  결과 공유 버튼            │
└─────────────────────────────────────────────────────┘
```

**레이아웃 (모바일):**
```
그래프 → 결과 카드 3종 (세로 스택) → 시나리오 비교
```

**핵심 인터랙션:**
- 첫 진입 시 결과 카드 숫자 0 → 목표값 카운트업 (1.2초)
- 그래프 라인 왼쪽 → 오른쪽 드로잉 애니메이션 (1.5초)
- 카드 3종 순차 페이드인 (stagger 0.15초)

---

### 6-3 설정 모달

**목적:** 대시보드에서 입력값을 수정하고 즉시 재계산한다.

**진입:** 대시보드 우상단 ⚙️ 버튼 → 배경 블러 + 모달 슬라이드인

**구성:**
```
기본 설정
  • 현재 나이
  • 월 저축액
  • 은퇴 희망 나이
  • 희망 월 인출액

▼ 고급 설정 (접이식 토글)
  • 기대 수익률 (기본값: 페르소나 프리셋)
  • 인플레이션율 (기본값: 2.5%)
  • 저축 증가율

[ 다시 계산하기 ]  → 모달 닫힘 + 그래프 즉시 업데이트
```

## 7. 기능 요구사항 (MoSCoW)

### 🔴 Must Have

#### F-01 희망 월 인출액 역산 계산기
- 순방향(자산→결과) 완전 제거, 역산 구조로 전환
- 슬라이더 조작 시 결과 실시간 업데이트 (debounce 300ms)
- "낙원까지 D-N년" 카운트다운 강조 표시

#### F-02 자산 성장 시각화 그래프
- 연도별 자산 라인 차트, 인터랙티브 호버 툴팁
- 은퇴 목표 시점 수직선 강조
- 슬라이더 조작 시 그래프 실시간 반응 (300ms)

#### F-03 인플레이션 반영 실질 구매력 표시
- 명목 자산 / 실질 구매력 동시 표시
- 기본값 2.5%, 사용자 조정 가능

#### F-04 페르소나 프리셋
- 온보딩 Section 5에서 "안정형 / 공격형" 선택
- 선택 시 수익률·저축증가율 자동 세팅, 이후 개별 조정 가능

| | 안정형 (이지현, 40세) | 공격형 (박준혁, 35세) |
|---|---|---|
| 수익률 | 4.5% | 9% |
| 저축 증가율 | 연 2% | 연 5% |
| 인출 전략 | 4% 룰 | 3.5% 룰 |
| 참고 모델 | 채권혼합형 펀드 | 글로벌 ETF 100% |

---

### 🟡 Should Have

#### F-05 시나리오 A/B 비교
- 두 시나리오를 같은 그래프 위에 동시 표시
- "시나리오 B가 N년 빠름 · 자산 N억 더 많음" 한 줄 요약
- 안정형 ↔ 공격형 프리셋을 기본 비교쌍으로 활용

#### F-06 "지금 시작 vs N년 후 시작" 비교
- "3년 미루면 월 인출 가능액이 87만원 줄어듭니다" 형태로 표시
- 타겟 심리("지금 시작해도 늦지 않을까") 정면 대응

#### F-07 결과 공유 (URL + 이미지)
- 입력값을 URL 파라미터로 인코딩 → 공유 링크 생성
- 결과 카드 이미지 다운로드 (카카오·SNS 최적화)

- F-09 세후 수익률 자동 계산 (금융소득세 15.4% 차감)
- F-10 몬테카를로 시뮬레이션 (목표 달성 성공 확률 표시)
