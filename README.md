# 🌴 낙원계산기 v2.0

**국내 유일의 시각적 FIRE 시뮬레이터**

사용자가 입력값을 바꾸면 자산 성장 그래프가 즉시 반응하는, 인터랙티브 FIRE(Financial Independence, Retire Early) 시뮬레이터입니다.

## 주요 기능

- **F-01** 자산 성장 그래프 — 연도별 라인 차트, 은퇴 시점 강조, 인터랙티브 호버
- **F-02** 슬라이더 UX — 실시간 그래프 업데이트 (debounce 300ms)
- **F-03** 결과 요약 카드 — 은퇴 연도 / 총 자산 / 월 인출액 / 낙원까지 D-N년
- **F-04** 시나리오 A/B 비교 — 2개 시나리오 동시 그래프
- **F-05** 인플레이션 토글 — 실질/명목 전환 (기본값 2.5%)
- **F-06** 결과 공유 — URL 파라미터 공유 링크 + 결과 이미지 다운로드

## 기술 스택

- Vanilla HTML / CSS / JavaScript (빌드 도구 없음)
- Chart.js 4.x (CDN)
- html2canvas (이미지 다운로드)

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다. 서버 불필요.

## 성능 목표

| 항목 | 기준 |
|------|------|
| 초기 로드 LCP | < 2.5초 (모바일 4G) |
| 그래프 반응 | 입력 변경 후 < 300ms |
| 데이터 저장 | 클라이언트 사이드 계산만 |
